About Shopp-E

Shopp-E is a full-stack online shopping application developed using Spring Boot for backend and Angular for frontend. It offers a seamless shopping experience with role-based access for users, retailers, and administrators.

Application Highlights

  • 🛒 User, Retailer, and Admin role-based access management.
  • 📦 Real-time product management (add, update, delete, view).
  • 💬 Wishlist and product comparison for smart shopping.
  • 📊 Admin dashboard for managing users and retailers.
  • ⚙️ Built with RESTful APIs ensuring fast and secure communication.
